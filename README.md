Welcome to VisionCamera V5.

If you see this, you got invited to the VisionCamera V5 project. Thank you for sponsoring me.

- A full rewrite of the Camera library to a more powerful connection-based API.
- Powered by [Nitro Modules](https://github.com/mrousavy/nitro).
- Multi-Cam Support (e.g. front + back camera at the same time)
- Depth Data Support (`Depth` frame processor + photo capture)
- More powerful AE/AF/AWB (focus/exposure/white-balance locking to distance, point or rect)
- More powerful Frame Processor APIs
- Stability Improvements
